# Vimbatech - Deployment Guide

## Free Hosting Options

### 1. Vercel (Recommended)
- Free tier available
- Easy deployment from GitHub
- Supports Python/Flask
- Automatic SSL certificate
- Custom domain support

### 2. Netlify
- Free tier available
- Static site hosting (for landing page)
- Can host Flask as serverless functions

### 3. GitHub Pages
- Completely free
- Best for static landing page
- Limited to static content

### 4. PythonAnywhere
- Free tier for Python web apps
- Built-in Flask support
- Limited resources
