# Loan Default Prediction Application

## Project Overview

This project implements a loan default prediction system using machine learning models. It is designed to analyze customer data and predict whether a loan will default, helping financial institutions make better lending decisions.

The key components include:

- Data processing and feature engineering scripts
- Model training scripts (not included in repo due to size)
- A Streamlit app (`loan_app.py`) to interactively predict loan default risk

## Important Note on Large Files

The trained model file `loan_default_model.joblib` is **excluded** from this repository due to its large size (over 1GB). Please do **not** attempt to upload it to GitHub.

Make sure to keep the model file in your local environment for running the Streamlit app.
